ยินดีต้อนรับเข้าสู่แฟ้มสะสมผลงานของนายเอกครินทร์ หงษ์วิเศษ
1.ผมเป็นคนชอบออกกำลังกาย(ฝืน)
2.ผมชอบทำอะไรที่ทำให้ไม่อึดอัด
3.ผมมักจะอ่านหนังสือและทำโจทย์บ่อยครั้ง
4.ผมมีความฝันที่จะเข้าจุฬา
5.เวลาว่างๆผมมักฟังเพลง
6.ผมเป็นคนไม่ค่อยพูดความรู้สึก
7.ผมชอบกินน้ำชา
8.อาหารที่ชอบก็เป็นข้าวผัด และกระเพรา
9.ผมเป็นคนไม่กล้าทักใครและไม่ค่อยกล้าทักเพื่อนใหม่ๆเวลาอยู่คนเดียว
10.ผมชอบเล่นกีฬา หลายอย่างแต่ไม่โดดเด่นสักอย่าง
